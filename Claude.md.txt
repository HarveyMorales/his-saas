## 🌐 ESTRATEGIA MULTIPLATAFORMA (OBLIGATORIO)

La app debe diseñarse con enfoque progresivo:

### Fase 1 — Web App (PRIORIDAD)
- Primera versión siempre web
- Rápida iteración y validación
- Accesible globalmente

### Fase 2 — Mobile App
- Migración a iOS y Android
- Uso de React Native o Flutter
- Reutilización máxima de lógica

### Fase 3 — Desktop (si aplica)
- Versión con Electron o similar
- Enfocada en power users

---

### 🧠 PRINCIPIO CLAVE

Diseñar desde el inicio para multiplataforma:
- Componentes reutilizables
- Lógica desacoplada del frontend
- API centralizada
